About the Project

What we built and how it works.

⛓️

On-Chain Check-ins

Every check-in is a transaction on Ethereum. Your visit history is stored permanently and can be verified by anyone.

🏙️

Multi-Facility Access

One subscription works at every registered facility in the network — gyms, pools, climbing walls and more.

💎

Referral Bonus

Refer a friend when they subscribe and both of you receive bonus credits on-chain — set by the admin and paid out automatically.

⚡

Peak Hour Pricing

Facilities can set a price multiplier for busy hours, stored on-chain. You see the exact cost before confirming a check-in.

🔓

No Middleman

When you check in, credits go directly to the facility's account. The smart contract handles the payment — no platform takes a cut.

🏅

VIP Badge

Visit the same facility 4 weeks in a row and earn a VIP badge — an on-chain ERC-1155 token that unlocks a check-in discount at that venue.
